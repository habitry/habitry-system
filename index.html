Welcome to the Habitry Source.

Habitry started out as the Health Psychology research of Steven M. Ledbetter, aka Coach Stevo, and then the became the basis of his practice as a coach. Since 2011, 36 cohorts with more than 500 clients have seen iterations of the Habitry System. But it was when Stevo started teaching it to more coaches that the pace of learning skyrocketed.

An Open System for Change

The mission of Habitry, Co. is to "help the most people help the most people." We believe the fastest way to get evidence-based behavior change to the broadest popultion is to open up the process of systematizing it. So Habitry, Co. has licenced these core teaching materials under Creative Commons Attribution 4.0 International. That means you can use all or part of this system to train your own clients, as long as you follow the rules of attribution. Mainly, if you are using Habitry, please say it's Habitry. If you are only using part of the system, you cannot call it "Habitry."

If you would like to learn more about what "Creative Commons" and "Open Source" means, here's a handy YouTube Video to explain it with Legos.

Using the System

What you'll see when you download the Habitry Source, is a living set of documents. The main document is a Tutorial on what Habitry is, why it works so well, and how you can implement it yourself. There are also supporting documents like templates for communicating with clients, research reviews on the foundational theories of Habitry, and the principles that flow from these theories and support all the iterations that make Habitry great. There is also an FAQ.

The Habitry Authors are constantly updating Habitry Source. It is not like a book, but more like a piece of software that is updated every time we learn about a bug or develop a new feature. We let users know about updates with regular emails, but if you want to be on the cutting edge we highly suggest becoming a Habitry Author.

Contributing to Source

We would not be able to expand on the Habtry Source at the rate that we have without the contribution of our Habitry Authors. Awesome coaches like Omar Ganai, Josh Hillis, Georgie Fear & Roland Fisher, Sean Flanagan, Dr. Spencer Nadolsky, and may more. If you would like to contribute to our Source, join Github and fork your own version of Habitry. If you think you have an awesome contribution, submit a pull request! Make a few awesome contributions that way, and we'll invite you to be an offcial Author and participate in future releases.
